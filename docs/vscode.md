# Setting Up Editor

### Project Settings

* 檔案 -> 喜好設定 -> 設定  
貼上以下設定 vscode 自動格式化程式碼 

```
{
  "prettier.eslintIntegration": true,
  "editor.formatOnSave": true
}
```
